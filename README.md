# PDF 公文資訊擷取工具

自動從 PDF 公文中擷取「主旨」、「發文機關」、「發文字號」，並整理輸出至 Excel。

## 下載 EXE

1. 點選上方 **Actions** 頁籤
2. 點選最新一筆 workflow
3. 在 **Artifacts** 區塊下載 `PDF公文資訊擷取工具`
4. 解壓縮後雙擊 `.exe` 即可使用（不需安裝 Python）

## 使用方式

1. 開啟程式
2. 點「選擇檔案」或「選擇資料夾」選取 PDF
3. 指定 Excel 輸出位置
4. 按「開始擷取」

## 支援欄位

| 欄位 | 範例 |
|------|------|
| 發文機關 | 教育部 |
| 發文字號 | 教育部臺教高字第1120089000號 |
| 主旨 | 請查照辦理… |
